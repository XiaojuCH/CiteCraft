# Lab Playbook: Evidence-Linked Research Deliverables

Teams adopt research tooling faster when the first demo produces a cited brief, a comparison matrix, and a slide deck that looks ready for a meeting.

## Practical guidance

Start from one fixed sample project so the story stays legible and every team member sees the same result.

Prefer deterministic pipelines before adding agent orchestration, because repeatability matters more than novelty in the first demo.

Show a trace panel that jumps from a claim back to the exact source chunk, even if the locator is only a page, section, or file path in P0.

## Risks

Overly broad automation reduces trust when users cannot explain where a statement came from.

PPTX fidelity can wait until slide content, speaker notes, and evidence links already feel strong.
